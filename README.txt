# 🎯 Box Click Challenge

A fun and interactive browser game built with **HTML, CSS, JavaScript, and jQuery**.

## 🚀 Features

* 3×3 game board
* Random box highlights every second
* Click the highlighted green box to score points
* Live score counter
* Smooth and responsive UI
* Built using jQuery events and animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* jQuery

## 🎮 How to Play

1. Click the **Start** button.
2. A random box turns **green** every second.
3. Click the green box before it changes.
4. Earn **1 point** for every correct click.
5. Try to achieve the highest score!

This project was created to practice jQuery concepts such as DOM manipulation, events, animations, timers, and interactive UI development.
